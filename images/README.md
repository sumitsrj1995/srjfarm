# Images Directory

Add your farm photos and product images here.

## Recommended Image Sizes

- **Product Images**: 800x600px (or 4:3 ratio)
- **Farm Photos**: 1200x800px (or 3:2 ratio)
- **Hero Images**: 1920x1080px (or 16:9 ratio)

## Image Formats

- Use **WebP** for best performance (with JPG fallback)
- Use **JPG** for photos
- Use **PNG** for logos/icons with transparency

## Optimization

Before uploading, optimize images:
- Compress using tools like TinyPNG, ImageOptim, or Squoosh
- Keep file sizes under 200KB for fast loading
- Use descriptive filenames: `fresh-milk-1l.jpg` not `IMG_001.jpg`
